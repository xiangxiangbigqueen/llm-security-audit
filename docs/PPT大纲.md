### Slide 3: 背景
- LLM 迅速发展，Agent 开始主导代码安全审计
- 现有系统：DeepAudit、AgentStalker、OpenSecurityAI、ESAA-Security
- 痛点：误报率高、缺乏验证、自动化程度不足
- 我们的目标：构建一个端到端、可验证、多 Agent 协同的安全审计系统

### Slide 4: 系统架构（图）
- 展示分层架构图
- 接入层 → 分析层 → 验证层 → 利用层 → 输出层

### Slide 5: 多 Agent 协同设计
- Scanner Agent（规则匹配 + LLM 分析）
- Verifier Agent（上下文 + SAST + LLM 三级验证）
- Reporter Agent（报告生成）
- Agent 间通信：通过 MCP 协议

### Slide 6: 关键技术 - SAST 扫描
- 8 类漏洞规则库（SQL注入、命令注入、XSS、SSRF等）
- 正则匹配 + 语义分析
- 上下文感知的误报过滤
- 多语言支持

### Slide 7: 关键技术 - LLM 深度分析
- Claude Code Agent 驱动的代码语义分析
- 识别传统规则无法发现的逻辑漏洞
- 自动生成攻击场景描述
- 提供精准的修复建议

### Slide 8: 关键技术 - PoC 自动生成
- 模板化生成各类型漏洞 PoC
- LLM 优化 PoC 代码质量
- 含复现步骤和验证方法
